_Publicado em 04 de novembro de 2025._

Bem-vindo! Este é o seu centro de comando para acessar e baixar as listas de empresas da **Base CNPJ**.

Para contextualizar o poder desta ferramenta: o portal abrange **TODAS as empresas ativas do país**. Para tornar essa imensidão de dados gerenciável, nós segmentamos tudo para você.

O resultado são mais de **60.000 arquivos** `.xlsx` prontos para download (totalizando 64.538 arquivos na data deste post), já organizados por Estado (UF), Regime Tributário e Atividade (CNAE).

Este guia vai te mostrar como encontrar exatamente o que precisa em segundos.

![Visão Geral do Portal de Downloads da Base CNPJ](portal-base-cnpj.png)

### 1. Como Acessar o Portal

Assim que seu pagamento for confirmado, você receberá um e-mail com o assunto **"Base CNPJ"**. Neste e-mail, você encontrará seu **link de acesso pessoal** e também um **código de acesso**.

> **Importante:** A forma mais fácil de entrar é usando o **link pessoal** do e-mail. Guarde-o, pois ele é sua chave de entrada, de uso pessoal e intransferível.

Caso você acesse o portal diretamente (sem usar o link pessoal), será solicitado o seu **código de acesso**. Nesse caso, basta copiar e colar o código que você recebeu no mesmo e-mail.

![Tela de Código de Acesso](portal-codigo-acesso.png)

### 2. Conhecendo a Tela

A interface é dividida em três áreas principais:

1.  **Barra de Filtros (Esquerda):** Onde você segmenta os dados por Regime e Estado.
2.  **Busca por CNAE (Topo):** O filtro mais poderoso. Permite pesquisar por atividade econômica.
3.  **Lista de Arquivos (Centro):** Onde os resultados aparecem prontos para baixar.

### 3. Como Encontrar Seus Arquivos (Os Filtros)

O segredo para encontrar os leads certos é combinar os filtros.

**Método 1: A Busca Rápida por CNAE (O Mais Usado)**

Esta é a forma mais direta de encontrar seu público.

* Use a barra de pesquisa **"Pesquise pelo código ou descrição da atividade (CNAE)"**.
* Você pode digitar o **código** (ex: `8610101` para "Atividades de atendimento hospitalar") ou
* Você pode digitar uma **palavra-chave** da descrição (ex: `hospitalar`, `soja`, `software`).

O portal filtrará instantaneamente todos os arquivos que correspondem àquela atividade.

**Método 2: Os Filtros de Segmentação (Barra Lateral)**

Use os filtros da esquerda para refinar sua busca:

* **COMPETÊNCIA:** Mostra o mês/ano de referência dos dados (ex: `10/2025`). Os dados são atualizados mensalmente.
* **REGIME:** Permite focar no tipo de empresa:
    * `MEI`: Microempreendedor Individual
    * `SIMPLES`: Empresas do Simples Nacional
    * `DEMAIS`: Outros regimes (Lucro Real, Presumido, etc.)
* **ESTADOS (UF):** Selecione um ou mais estados para focar sua prospecção regional.

> **Exemplo de combinação:**
> 1.  Digite `padaria` na **Busca por CNAE**.
> 2.  Selecione `MEI` no filtro **REGIME**.
> 3.  Selecione `SP` no filtro **ESTADOS (UF)**.
> *Pronto! A lista de arquivos mostrará todas as planilhas de MEIs do estado de SP que são padarias.*

### 4. Como Baixar os Arquivos

Você tem duas formas de fazer o download:

1.  **Download Individual:** Simplesmente clique no nome do arquivo `.xlsx` desejado.
2.  **Download em Lote (Recomendado):**
    * Marque as caixas de seleção dos arquivos que você quer.
    * Para marcar todos os arquivos que aparecem na sua tela (após aplicar os filtros), clique em **"Selecionar todos"**.

> **⚠️ ATENÇÃO: Limite de Download**
> Para garantir a estabilidade do servidor para todos os usuários, os downloads em lote são limitados a **100 arquivos por vez**.


### 5. Entendendo o Nome do Arquivo

Os nomes dos arquivos são padronizados para você saber o que está baixando antes mesmo de abrir:

`[Competência]_[Regime]_[UF]_[CNAE]_[Parte].xlsx`

**Exemplo:** `202510_DEMAIS_AC_0111301_arq1.xlsx`
* **202510:** Dados de Outubro de 2025.
* **DEMAIS:** Regime "Demais".
* **AC:** Estado do Acre.
* **0111301:** CNAE "Cultivo de Soja".
* **arq1:** Arquivo parte 1 (dividimos arquivos muito grandes em partes para facilitar o download).
